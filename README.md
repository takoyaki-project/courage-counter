# 次行こか

飛び込み営業応援ツール

![メイン画像](images/main.png)

---

## 🌟 これは何？

**次行こか** は、飛び込み営業や新規開拓営業を頑張る人のための無料Webアプリです。

断られた回数や、話を聞いてもらえた回数をワンタップで記録できます。

営業活動を前向きに続けるための「行動の見える化ツール」です。

## ✨ できること

* 断られた回数をワンタップで記録
* 話を聞いてもらえた回数を記録
* 月ごとの履歴を自動保存
* 前向きな応援メッセージを表示
* スマホで手軽に利用可能

---

## 📱 使い方

1. 営業訪問後にアプリを開く
2. 「断られた！」または「話を聞いてもらえた！」をタップ
3. 月ごとの活動履歴を確認する
